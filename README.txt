My dissertation aimed at tackling a technology called Node.js and the goal was to build a chat application for children. The duration of this project was one month with no prior knowledge of the subject.


*********************************** GET STARTED ***********************************

1. Run the command npm i to install all the packages needed
2. Open the Command Prompt and navigate to where your mongodb folder is setup
3. In the Command Prompt - type 'mongod'
4. Open another Command Prompt and navigate to the same folder and type 'mongo'
5. To change to the correct database, enter the mongodb command 'use examen' 
5. Open another Command Prompt and navigate to where you cloned the project
6. Type 'node app.js'
7. Open the browser and enter the url: 'localhost:1338'
